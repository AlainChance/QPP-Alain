# QPP-Alain
This repository contains companion software of the paper "Quantum Permutation Pad with Qiskit Runtime" by Alain Chancé published with ID CS23-229 in the ICCCAS 2023 Conference Papers Collections. You can download it via the following link: http://www.icccas.org/download/ICCCAS2023-Paper-Collection.zip

Check out a recording of Alain's speech here: https://www.youtube.com/watch?v=aUnWFgz-vio

Dr. Randy Kuang and Prof. Michel Barbeau have chaired Session 6, Quantum Information, http://www.icccas.org/Track-4.html, at the 12th Edition of the International Conference on Communications, Circuits, and Systems (ICCCAS) in Singapore from May 5-7, 2023.

Michel Barbeau's profile: https://carleton.ca/scs/people/michel-barbeau/
